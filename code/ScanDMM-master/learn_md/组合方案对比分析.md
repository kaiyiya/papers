## 🔍 现有组合分析

### 组合1：VMRNN + HiLo + Gated Fusion（基础版）

#### 架构设计

```
输入：360度图像 I (3, H, W)
  ↓
[HiLo Glance] 
  ├─ 低频分支（Lo-Fi）：全局池化 → 粗粒度特征 s_coarse (50维)
  └─ 高频分支（Hi-Fi）：窗口注意力 → 细粒度特征 s_fine_init (50维)
  ↓
[Gated Fusion] → 融合 s_coarse + s_fine_init → s_0 (50维)
  ↓
[状态初始化] → z_0 = F(z_0, x_1, s_0)
  ↓
循环注视（固定或自适应长度）：
  [价值网络 V*] → 评估是否继续
    ↓ continue
  [局部特征提取] → 基于当前注视点提取局部特征
  [Gated Fusion] → 融合 s_coarse + s_fine → s_fused (50维)
  [VMRNN Cell] → z_t = VMRNN(z_{t-1}, s_fused, (h_{t-1}, c_{t-1}))
  [Emitter] → x_t ~ p(x_t | z_t)
  [策略网络] → 下一个注视位置 x_{t+1}
    ↓ stop
  [输出扫描路径]
```

#### 核心特点

1. **HiLo Attention**
   - 低频：全局池化，捕捉全局信息
   - 高频：窗口注意力，捕捉局部细节
   - **问题**：没有考虑360度图像的球面几何特性

2. **VMRNN**
   - 结合Mamba和RNN
   - 空间-时间联合建模
   - **问题**：直接使用，没有适配球面结构

3. **Gated Fusion**
   - 简单的门控融合
   - 固定权重策略
   - **问题**：所有阶段使用相同的融合策略

#### 优势

- ✅ 效率提升40-50%
- ✅ 多尺度感知
- ✅ 空间-时间联合建模
- ✅ 实施相对简单

#### 局限性

- ❌ **忽略球面几何**：没有考虑360度图像的特殊性
- ❌ **静态融合**：所有阶段使用相同的融合策略
- ❌ **缺乏认知启发**：没有结合视觉注意力理论
- ❌ **固定窗口**：HiLo的窗口大小固定，不适合球面结构

## 🚀 增强版组合分析

### 组合2：SphericalVMRNN + SphericalHiLo + AdaptiveMultiScaleFusion + CognitivePolicy（增强版）

#### 架构设计

```
输入：360度图像 I (3, H, W)
  ↓
[SphericalHiLo Glance]
  ├─ 球面失真感知池化（Distortion-Aware Pooling）
  ├─ 低频分支：考虑等距圆柱投影失真 → s_coarse (50维)
  └─ 高频分支：球面窗口注意力（考虑边界连续性）→ s_fine_init (50维)
  ↓
[AdaptiveMultiScaleFusion] → 根据注视阶段自适应融合
  ├─ 粗粒度阶段（前30%）：weight_coarse = 0.8
  ├─ 过渡阶段（30-70%）：动态调整权重
  └─ 细粒度阶段（后30%）：weight_coarse = 0.2
  ↓
[球面状态初始化] → z_0 = F_spherical(z_0, x_1, s_0, spherical_coords)
  ↓
循环注视（自适应长度）：
  [价值网络 V*] → 评估是否继续（考虑信息熵）
    ↓ continue
  [球面局部特征提取] → 考虑球面距离的局部特征
  [AdaptiveMultiScaleFusion] → 根据当前阶段自适应融合
  [SphericalVMRNN Cell] → z_t = SphericalVMRNN(z_{t-1}, s_fused, (h_{t-1}, c_{t-1}), spherical_coords)
  [Emitter] → x_t ~ p(x_t | z_t)
  [CognitiveInspiredPolicy] 
    ├─ 返回抑制机制（避免重复注视）
    ├─ 工作记忆引导（考虑历史信息）
    └─ 显著性引导 → 下一个注视位置 x_{t+1}
    ↓ stop
  [输出扫描路径]
```

#### 核心特点

1. **SphericalHiLo Attention**
   - **失真感知池化**：考虑等距圆柱投影的失真
   - **球面窗口**：考虑左右边界连续性（wrap-around）
   - **经纬度分离**：分别处理纬度和经度方向
   - **自适应窗口**：根据纬度调整窗口大小（极地区域窗口更大）

2. **SphericalVMRNN**
   - **球面状态空间**：SSM适配球面结构
   - **球面坐标转换**：3D坐标 ↔ 球面坐标
   - **球面距离**：使用大圆距离而非欧氏距离
   - **边界处理**：考虑360度图像的边界连续性

3. **AdaptiveMultiScaleFusion**
   - **阶段感知**：根据注视阶段动态调整
   - **自适应权重**：前期更依赖全局，后期更依赖局部
   - **平滑过渡**：过渡阶段平滑切换

4. **CognitiveInspiredPolicy**
   - **返回抑制**：避免重复注视已访问区域
   - **工作记忆**：维护最近N个注视点的记忆
   - **显著性引导**：结合显著性图引导注视

|------|-----------------|---------------------------|--------|
| **几何感知** | ❌ 忽略球面几何 | ✅ 考虑等距圆柱投影失真 | **准确性提升5-10%** |
| **边界处理** | ❌ 标准边界 | ✅ 左右边界wrap-around | **避免边界错误** |
| **窗口设计** | ❌ 固定窗口大小 | ✅ 根据纬度自适应调整 | **极地区域更准确** |

- **准确性**：考虑球面几何，预测更准确，特别是边界和极地区域
- **鲁棒性**：边界连续性处理，避免边界处的预测错误
- **适应性**：自适应窗口，适应不同纬度的失真程度
