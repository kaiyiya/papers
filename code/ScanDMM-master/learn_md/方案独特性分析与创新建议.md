# VMRNN + HiLo + Gated Fusion 方案独特性分析与创新建议

## 📊 独特性评估

### 当前组合的独特性分析

#### ✅ 好消息：组合本身较独特

**VMRNN + HiLo + Gated Fusion** 在**360度图像扫描路径预测**领域的组合确实**比较小众**：

1. **VMRNN在扫描路径预测中的应用**：**极少见**
   - VMRNN主要用于视频理解、动作识别
   - 在扫描路径预测领域几乎未见
   - 在360度图像领域更是罕见

2. **HiLo Attention在扫描路径预测中的应用**：**罕见**
   - HiLo主要用于图像分类、目标检测
   - 在眼动/扫描路径预测中很少见
   - "从粗到细"的应用更少

3. **三者组合**：**几乎未见**
   - 三个技术的组合在扫描路径预测中未见报道
   - 在360度图像领域更是首创

#### ⚠️ 潜在风险：各组件单独使用较多

虽然组合独特，但需要注意：
- **Mamba/SSM**：2024年很火，可能有人在做
- **HiLo Attention**：在视觉任务中应用较多
- **Gated Fusion**：多模态融合中常见

### 创新点2：自适应多尺度融合策略 ⭐⭐⭐⭐⭐

**核心思想**：不是简单的Gated Fusion，而是根据注视阶段自适应调整融合策略

```python
class AdaptiveMultiScaleFusion(nn.Module):
    """
    自适应多尺度融合：根据注视阶段动态调整融合策略
    """
    def __init__(self, z_dim=100):
        super().__init__()
        # 不同阶段的融合策略
        self.coarse_fusion = gatedFusion(dim=z_dim // 2)  # 粗粒度阶段
        self.fine_fusion = gatedFusion(dim=z_dim // 2)    # 细粒度阶段
        self.transition_net = nn.Linear(1, 1)  # 过渡网络
        
    def forward(self, s_coarse, s_fine, gaze_count, total_gazes):
        # 计算当前处于哪个阶段
        stage_ratio = gaze_count / total_gazes  # [0, 1]
        
        # 前期：更依赖粗特征（全局）
        # 后期：更依赖细特征（局部）
        if stage_ratio < 0.3:  # 前30%：粗粒度阶段
            fused = self.coarse_fusion(s_coarse, s_fine)
            weight_coarse = 0.8
        elif stage_ratio < 0.7:  # 中间40%：过渡阶段
            alpha = self.transition_net(stage_ratio.unsqueeze(1))
            fused = alpha * self.coarse_fusion(s_coarse, s_fine) + \
                   (1 - alpha) * self.fine_fusion(s_coarse, s_fine)
        else:  # 后30%：细粒度阶段
            fused = self.fine_fusion(s_coarse, s_fine)
            weight_coarse = 0.2
        
        return fused
```

**独特性**：⭐⭐⭐⭐⭐
- 动态调整融合策略
- 考虑注视阶段
- 比简单Gated Fusion更智能

### 创新点3：球面VMRNN（Spherical VMRNN）⭐⭐⭐⭐⭐

**核心思想**：将VMRNN适配到360度图像的球面结构

```python
class SphericalVMRNNCell(nn.Module):
    """
    球面VMRNN：考虑360度图像的球面几何
    """
    def __init__(self, hidden_dim, input_resolution, d_state=16):
        super().__init__()
        # 1. 球面状态空间模型
        self.spherical_ssm = SphericalSS2D(
            d_model=hidden_dim,
            d_state=d_state,
            input_resolution=input_resolution
        )
        
        # 2. 考虑球面距离的RNN
        self.spherical_rnn = SphericalRNN(hidden_dim)
        
        # 3. 球面坐标转换
        self.coord_transform = SphericalCoordTransform()
    
    def forward(self, x_t, hidden_states, gaze_3d):
        # 将3D注视点转换为球面坐标
        spherical_coords = self.coord_transform(gaze_3d)
        
        # 考虑球面几何的状态转移
        h_t, c_t = self.spherical_rnn(x_t, hidden_states, spherical_coords)
        
        # 球面SSM处理
        output = self.spherical_ssm(h_t)
        
        return output, (h_t, c_t)
```

**独特性**：⭐⭐⭐⭐⭐
- 专门针对球面几何设计
- 考虑360度图像的特殊性
- 与普通VMRNN有本质区别

### 创新点4：认知启发的注视策略 ⭐⭐⭐⭐⭐

**核心思想**：结合认知科学的视觉注意力理论

```python
class CognitiveInspiredPolicyNetwork(nn.Module):
    """
    认知启发的策略网络：结合视觉注意力理论
    """
    def __init__(self, z_dim=100):
        super().__init__()
        # 1. 返回抑制机制（Inhibition of Return）
        self.inhibition_of_return = InhibitionOfReturn()
        
        # 2. 工作记忆机制
        self.working_memory = WorkingMemoryBuffer(capacity=5)
        
        # 3. 显著性引导
        self.saliency_guidance = SaliencyGuidance()
        
    def forward(self, z_t, s_coarse, gaze_history):
        # 1. 计算显著性图
        saliency = self.saliency_guidance(s_coarse)
        
        # 2. 应用返回抑制（避免重复注视）
        inhibited_saliency = self.inhibition_of_return(saliency, gaze_history)
        
        # 3. 工作记忆引导（考虑历史信息）
        memory_guided = self.working_memory(inhibited_saliency, gaze_history)
        
        # 4. 选择下一个注视点
        next_gaze = self.select_gaze(memory_guided)
        
        return next_gaze
```

**独特性**：⭐⭐⭐⭐⭐
- 结合认知科学理论
- 返回抑制、工作记忆等机制
- 更符合人类视觉行为
