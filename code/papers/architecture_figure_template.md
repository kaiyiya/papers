# MambaGaze 架构图绘制模板与指南

## 📐 推荐工具

### 1. **PowerPoint (推荐) ⭐⭐⭐⭐⭐**
**优点：**
- ✅ 易于使用，上手快
- ✅ 对齐和布局工具完善
- ✅ 支持矢量图形，导出PDF质量高
- ✅ 可以轻松调整颜色、字体、箭头
- ✅ 支持组合和分组，便于整体移动
- ✅ 可以导出为PDF、PNG（高分辨率）

**设置建议：**
- 页面大小：A4横向 或 自定义 16:9 (宽屏)
- 分辨率：导出时选择300 DPI（高质量）
- 字体：使用论文中的字体（如Times New Roman, Arial）

### 2. **Draw.io (diagrams.net) ⭐⭐⭐⭐**
**优点：**
- ✅ 免费在线工具
- ✅ 专业流程图绘制
- ✅ 支持LaTeX数学公式
- ✅ 可以导出为PDF、PNG、SVG

**缺点：**
- 需要学习曲线
- 在线版本需要网络

### 3. **Adobe Illustrator / Inkscape ⭐⭐⭐**
**优点：**
- ✅ 专业矢量图形工具
- ✅ 最高质量输出

**缺点：**
- 学习曲线陡峭
- 可能过于复杂

## 🎨 架构图布局设计

### 整体布局：**三列横向布局**

```
┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│  [左列]              [中列]                    [右列]                  │
│  Glance Network     Hierarchical Sampler      Mamba + Decoder          │
│  (全局特征提取)      + Focus Network          (序列建模)               │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 尺寸建议
- **宽度**：约 16-18 cm（适合论文双栏）
- **高度**：约 8-10 cm
- **比例**：约 2:1（宽:高）

## 📐 PowerPoint 绘制步骤

### Step 1: 设置画布
1. 新建幻灯片
2. 设计 → 幻灯片大小 → 自定义
3. 宽度：40 cm，高度：20 cm（或按需调整）
4. 背景：白色

### Step 2: 绘制基础框架
1. 插入 → 形状 → 圆角矩形
2. 绘制三个大框（左、中、右列）
3. 设置填充色（浅色）和边框（深色，2pt）

### Step 3: 绘制组件
1. **Glance Network**：
   - 绘制多个小矩形表示Conv Block
   - 用特殊颜色标注CoordAttention
   - 添加文字标注

2. **Hierarchical Sampler**：
   - 绘制决策框（Phase 1/Phase 2）
   - 添加公式或文字说明
   - 用流程图形状表示采样过程

3. **Focus Network**：
   - 类似Glance Network的结构
   - 用不同颜色区分

4. **Mamba Block**：
   - 绘制一个大框
   - 内部可以画简化的SSM示意图
   - 标注关键参数

### Step 4: 添加连接线
1. 插入 → 形状 → 箭头
2. 使用实线箭头表示数据流
3. 使用虚线箭头表示可选路径（如teacher forcing）
4. 箭头颜色：深灰色（#595959），粗细：1.5pt

### Step 5: 添加标注
1. 使用文本框添加数学符号
   - 插入 → 公式（或使用LaTeX插件）
   - 或使用Unicode符号：ℝ, ∈, ←, →, ↑, ↓
2. 字体：Times New Roman 或 Arial
3. 字号：8-10pt（根据整体大小调整）

### Step 6: 美化
1. 对齐工具：格式 → 对齐
2. 统一间距
3. 添加阴影（可选，轻微）
4. 检查颜色对比度

### Step 7: 导出
1. 文件 → 另存为 → PDF
2. 或：文件 → 导出 → 更改文件类型 → PNG
3. PNG分辨率：300 DPI

## 📝 关键标注内容

### 必须包含的标注：

1. **输入**：
   - 360° Image I (3×H×W)
   - Saliency Map S (1×H×W)

2. **关键参数**：
   - d = 256
   - ε = 0.60 / 0.30
   - α = 0.4
   - τ = 0.12
   - μ = 0.45
   - s_max = 0.18

3. **维度标注**：
   - g ∈ ℝ^256
   - f_t ∈ ℝ^256
   - h_t ∈ ℝ^256
   - p̂_t ∈ [0,1]²

4. **关键公式**（简化版）：
   - h_t = g + f_t + MLP_pos(p_t)
   - p̂_t = σ(MLP_dec(Mamba(H)_t))
