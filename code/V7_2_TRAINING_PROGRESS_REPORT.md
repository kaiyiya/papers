## 🔍 V7.2核心改进

### 1. 改进的覆盖率损失函数
```python
class ImprovedCoverageLoss(nn.Module):
    def __init__(self, x_weight=2.0, y_weight=1.0):
        # X方向权重2倍于Y方向
        self.x_weight = x_weight
        self.y_weight = y_weight
    
    def forward(self, scanpaths):
        # 同时优化X和Y方向
        x_loss = -(x_std + x_range) / 2.0
        y_loss = -(y_std + y_range) / 2.0
        total_loss = self.x_weight * x_loss + self.y_weight * y_loss
```

### 2. 调整的超参数
- **覆盖率损失权重**: 0.1 → 0.05 (降低50%)
- **探索率**: 70%/25% → 60%/30% (更平衡)
- **Checkpoint频率**: 每10个epoch → 每5个epoch
