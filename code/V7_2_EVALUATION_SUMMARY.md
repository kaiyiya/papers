## 🔍 关键发现

### 1. 训练集表现优异
- Salient360上达到或超越人类水平
- 说明模型架构和训练策略有效

### 2. 泛化能力不均衡
- Sitzmann表现优秀，AOI和JUFE表现不佳
- 可能原因：
  - 数据集特征差异较大
  - 训练数据覆盖不足
  - 模型对某些场景过拟合

### 3. REC指标稳定
- 所有数据集上REC都在4.5-6.5%
- 说明模型能够捕捉合理的回访模式
- 这是一个重要的优势

### 4. 覆盖率符合设计
- X覆盖: 65-77% ✅
- Y覆盖: 26-33% ✅
- X/Y比例: 2.6-3.0 ✅
